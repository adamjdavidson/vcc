# Design History

**Purpose:** Historical design briefs preserved for maintainers. **The LLM should skip this folder entirely.**

This folder holds the design briefs that originally shaped the printed worksheet packet and this repository. They are preserved as historical reference for maintainers.

**The LLM should skip this folder.** None of these documents describe how the kit currently works. Operational instructions live in `START_HERE_FOR_LLM.md`, `ai-master-instructions.md`, and `REPO-MAP.md`.

## Files

- `worksheet-design-brief.md` — the standalone brief used to design the printed worksheet packet. The packet now exists as `cheese_worksheets.pdf` at the kit root and is titled *Tell Your Story* (the brief proposed *Make the Value Travel*).
- `repo-build-brief.md` — the original brief used to build this repository. Some folder names ended up different from the brief's proposal (the kit uses `story-principles/`, `workflow/`, `research/` rather than `principles/`, `intake/`, `research-guides/`).

## When this folder is useful

- A future maintainer trying to understand original design intent.
- A future contributor pitching a redesign and wanting to see what the kit was designed against.
- Adam himself, six months from now, trying to remember why a given decision was made.

When this folder is *not* useful: when an LLM is helping a cheesemaker. Skip it.
