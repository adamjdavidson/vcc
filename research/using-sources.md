# Using Sources

**Purpose:** Source hierarchy and citation discipline for the LLM.

This repository should help the LLM research, but it should not become a machine for making unsupported claims.

## Source hierarchy

For claims about the specific cheese:

1. Cheesemaker worksheet, voice memo, or interview.
2. The cheesemaker’s own website or product sheet, if provided.
3. Directly relevant outside source about the maker or product.
4. General cheese-style or place research.

General research is useful context, but it is not proof about a specific cheese.

## Citation behavior

When using outside research, cite sources if the interface allows.

At minimum, name the source and keep research separate from maker-provided facts.

## If sources conflict

If the origin of a cheese style is disputed, say so.

If sources disagree about Vermont history or local facts, say so.

Do not smooth over uncertainty for narrative convenience.

## Research output format

Use this format when presenting research:

```markdown
## What you told me
...

## Outside context I found
...

## Possible story connections
...

## What I need you to confirm
...
```

This keeps the story truthful and prevents accidental fabrication.
