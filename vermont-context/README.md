# Vermont Context

**Purpose:** Background for location and regional storytelling. Vermont is not a story by itself.

These files provide background for location and regional storytelling.

Use them as context, not as automatic copy.

Vermont is not a story by itself. “Made in Vermont” is a starting point, not a finished narrative.

The LLM should use exact location, milk source, farm, creamery, animals, season, weather, history, and market role to make the story more specific.

Always avoid generic Vermont tourist language unless the facts justify it.
