# Story Options Output

**Purpose:** Format for presenting three possible story through-lines to the cheesemaker before drafting final copy.

Use this format before drafting final copy.

```markdown
## Strongest raw material
- ...

## Useful outside context
- ...

## Possible story through-line 1: [name]
One-sentence version:

Supporting facts:

Best audience:

What needs confirmation:

## Possible story through-line 2: [name]
...

## Possible story through-line 3: [name]
...

## What is missing
- ...

## Five follow-up questions
1. ...
2. ...
3. ...
4. ...
5. ...
```

Through-lines should be narrative possibilities, not isolated selling points.
