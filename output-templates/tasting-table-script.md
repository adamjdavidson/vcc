# Tasting Table Script

**Purpose:** Template for a market table or tasting script — natural, flexible, sounds like a person talking.

A tasting table script should be natural and flexible.

It should help a maker or staff member speak about the cheese without sounding rehearsed.

## Structure

1. Invite the taste.
2. Name the cheese plainly.
3. Offer one story detail.
4. Tell the taster what to notice.
5. Offer one use or pairing.

## Template

> Try this one next. It’s our [plain category]. [One story detail.] When you taste it, notice [sensory cue]. I like it with [pairing/use] because [reason].

## Add-ons

If the person is curious, add:

- cheese-style background,
- maker story,
- place story,
- process detail,
- price explanation.

## Rule

The first sentence should make tasting easy. The deeper story can follow if the person leans in.
